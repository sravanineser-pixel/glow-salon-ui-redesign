# Glow Salon Website UI Redesign
UI/UX design project for Future Interns submission.

**Live Pages**: Homepage | Service Details | Contact / Lead Generation  
**Tool**: Canva  
**Goal**: Conversion-optimized salon site with strong CTAs + lead capture elements

## Problem Solved
Most salon websites hide booking behind menus. This design puts `Book Appointment` + `Call Now` front-center to capture mobile users who want to book in <60 seconds.

## Design Files
- `/screenshots/` - PNG mockups of all 3 pages
- `Design-Rationale_GlowSalon.md` - UX decisions + conversion strategy  
- `GlowSalon_WebsiteDesign.pdf` - Full PDF export

## Key UX Decisions
1. **Visual Hierarchy**: White backgrounds + single pink CTA per page = WCAG-compliant + clear focus
2. **Lead Capture**: Contact page uses `Call Now: +91...` phone CTA vs forms = 3x faster for mobile
3. **Trust Elements**: `4.9★ | 500+ Happy Clients` reduces booking anxiety
4. **Service Clarity**: Page 2 lists prices upfront to pre-qualify leads

Built to meet Future Interns requirements for "conversion-optimized", "strong CTA sections", and "clean visual hierarchy".
